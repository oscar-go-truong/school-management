## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/oscar-go-truong/school-management.git
   ```
2. Install composer packages
   ```sh
   composer install
   ```
3. start the server
   ```sh
   php artisan serve
   ```
